# 🚀 운동 유형 테스트 배포 가이드

총 3단계, 약 10분이면 끝나요!

---

## 1단계: GitHub 가입 + 코드 올리기 (5분)

### GitHub 가입
1. **https://github.com** 접속
2. **Sign up** 클릭
3. 이메일, 비밀번호, 유저네임 입력 → 가입 완료

### 코드 올리기
1. 로그인 후 오른쪽 상단 **+** 버튼 → **New repository** 클릭
2. Repository name: `workout-type-test` 입력
3. **Public** 선택
4. **Create repository** 클릭

### 파일 업로드
1. 방금 만든 저장소 페이지에서 **uploading an existing file** 클릭
2. 다운받은 **workout-type-test.zip** 압축을 풀기
3. 압축 푼 폴더 안의 **모든 파일**을 드래그 앤 드롭으로 업로드
   - ⚠️ `node_modules` 폴더는 제외! (없을 거예요)
   - 올라가야 할 파일들: `package.json`, `vite.config.js`, `index.html`, `src/` 폴더 등
4. **Commit changes** 클릭

---

## 2단계: Vercel 가입 + 연결 (3분)

1. **https://vercel.com** 접속
2. **Sign Up** → **Continue with GitHub** 클릭
3. GitHub 계정으로 로그인 허용

---

## 3단계: 배포하기 (2분)

1. Vercel 대시보드에서 **Add New** → **Project** 클릭
2. **Import Git Repository** 에서 `workout-type-test` 저장소가 보일 거예요 → **Import** 클릭
3. 설정 화면이 나오면 **아무것도 바꾸지 말고** 그대로 **Deploy** 클릭
4. 1~2분 기다리면 배포 완료! 🎉

---

## 완료! 🎊

배포가 끝나면 이런 URL을 받아요:
```
https://workout-type-test.vercel.app
```

이 URL을 인스타 스토리, 카톡 등에 공유하면 됩니다!

## 추가 팁

### 커스텀 도메인 연결하고 싶다면
1. Vercel 프로젝트 → **Settings** → **Domains**
2. 원하는 도메인 입력 (예: `workout-test.kr`)
3. DNS 설정 안내를 따르면 끝

### 수정사항이 생기면
1. GitHub에서 파일을 직접 수정하거나, 새 파일 업로드
2. Vercel이 자동으로 감지해서 다시 배포해줘요 (약 1분)
